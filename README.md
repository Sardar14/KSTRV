# KSTRV1
A Scene Text Recognition (STR) Dataset for Central Kurdish in (Arabic-Based) Script.
KSTRV1 is a STR dataset comprises 1,420 natural scene images and 19,872 cropped word samples, covering Kurdish (Sorani and Badini dialects), Arabic, and English.
